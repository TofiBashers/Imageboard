Imageboard
==========
An imageboard is a type of Internet forum which operates mostly via posting images.

Features:  
-anonymity  
-attachment files (only images)  

Imageboard samples:   
http://www.4chan.org/  
http://krautchan.net/  
https://2ch.hk/  

Java version: 1.7  
Database: MySQL  

Spring Framework 3.2.0
MVC
JPA (Hibernate provider)

Frontend: HTML, CSS, jQuery
